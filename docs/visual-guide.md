# 画面操作ガイド

このガイドは現在の UI に合わせた操作イメージです。

## 1. ログイン画面

```
ITSM Incident Tracker

ログイン
ユーザー名 [                    ]
パスワード [                    ]
[ログイン]

または
[GitHub でログイン]
```

### 操作

| 操作 | 結果 |
|---|---|
| ユーザー名とパスワードを入力して `ログイン` | セッション Cookie が発行され、トップページへ移動 |
| `GitHub でログイン` | GitHub 認可画面へ移動 |
| OAuth エラー発生 | ログイン画面にエラーメッセージを表示 |

## 2. トップページ

```
ITSM Incident Tracker                         user role [ログアウト] [インシデント起票]

[総インシデント] [未解決] [P1 Active] [SLA応答違反] [SLA解決違反] [平均解決時間]

[ new ] [ assigned ] [ in_progress ] [ resolved ] [ closed ]

検索 [タイトル・説明・報告者で検索...]  優先度 [すべての優先度]  [更新]
admin の場合: [削除済みを表示]  [N 件を削除]

□ ID        タイトル        優先度    ステータス     SLA(解決)   起票者   担当者   起票日
□ INC-0001  ログイン不可    P1        New            3h 20m      admin    admin    2026/05/12 09:00
```

### 操作

| 操作 | 結果 |
|---|---|
| `インシデント起票` | 起票ダイアログを開く |
| ステータスボタン | 一覧を単一ステータスで絞り込み。再クリックで解除 |
| 検索入力 | タイトル、説明、起票者で検索 |
| 優先度選択 | 優先度で絞り込み |
| 更新アイコン | KPI と一覧を再取得 |
| 行クリック | 詳細画面へ移動 |
| チェックボックス | admin のみ削除対象を選択 |
| `削除済みを表示` | admin のみ削除済みだけを表示 |

## 3. 起票ダイアログ

```
インシデント起票

タイトル *  [                                      ]
説明 *      [                                      ]
            [                                      ]

優先度 [P3 - Medium]     担当者 [admin]

[キャンセル] [起票する]
```

### 入力項目

| 項目 | 内容 |
|---|---|
| タイトル | 必須。短い概要 |
| 説明 | 必須。状況や影響範囲 |
| 優先度 | P1 Critical / P2 High / P3 Medium / P4 Low |
| 担当者 | 自由入力。初期値はログインユーザー |

起票に成功するとダイアログが閉じ、一覧が更新されます。

## 4. 詳細画面

```
[戻る] INC-0001 | P1 | New | 応答SLA | 解決SLA

ログインシステムが応答しない [編集]
起票者 admin   起票日時 2026/05/12 09:00

説明
午前9時ごろからログインが失敗しています。

SLA
応答期限 05/12 09:15    解決期限 05/12 13:00

タイムライン
- admin 05/12 09:00 ステータスを変更: - -> New

更新
更新者 *   [admin]
ステータス [New]
優先度     [P1 - Critical]
担当者     [admin]
コメント   [作業メモ]
[更新する]

クイック遷移
[-> Assigned]
[-> In Progress]
[-> Closed]
```

### 操作

| 操作 | 結果 |
|---|---|
| 戻るボタン | トップページへ戻る |
| タイトルの編集アイコン | タイトルをインライン編集 |
| `更新する` | 変更フィールドとコメントを監査ログへ記録 |
| クイック遷移 | 推奨ステータスに即時更新 |
| コメントのみ入力して更新 | コメントとしてタイムラインへ記録 |

## 5. SLA 表示

| 表示 | 意味 |
|---|---|
| `3h 20m` | 期限まで余裕あり |
| `25m` | 期限 30 分前を切った警告状態 |
| `-40m` | 期限超過 |
| `応答済` | 初回応答済み |
| `SLA達成` | 解決済み |

## 6. admin 専用操作

```
[削除済みを表示]
□ INC-0003 削除済  タイトル取り消し線 ...
```

- 通常表示では削除済みを除外します。
- `削除済みを表示` をオンにすると削除済みのみ表示します。
- 削除済み行は半透明で、詳細画面には遷移しません。
- 復元機能はありません。
